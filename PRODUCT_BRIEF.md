# CargoTrack — Product Brief

CargoTrack is a B2B SaaS platform for real-time cargo shipment tracking, ML-powered delay prediction, and logistics management across East Africa's Northern Corridor.

**Product:** CargoTrack  
**Company:** CargoTrack Ltd  
**Target users:** Freight forwarders, 3PLs, importers/exporters  
**Stack:** Django REST Framework (API) + React + Tailwind CSS (frontend) + React Native (mobile)  
**Tone:** Professional, enterprise-grade
